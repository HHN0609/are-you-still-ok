Are You OK is a personal safety guardian application designed to ensure the well-being of users living alone or those who want an extra layer of security. The app requires users to "Check In" periodically. If a check-in is missed within a set "Golden Time", the system can trigger alerts or reminders (conceptually), helping to ensure timely assistance in case of emergencies.

*   **Daily Check-in**: Simple one-tap check-in mechanism to confirm safety.
*   **Golden Hours**: Customizable safe time intervals (e.g., every 24 hours).
*   **Status Monitoring**: Visual indicators for "Safe" (Checked in) and "Overdue" (Red alert status).
*   **Emergency Contact**: Register emergency contacts for critical situations.
*   **Modern UI**: Clean, Material Design interface with smooth animations and responsive feedback.
